# Task: Short Character Description

Write a brief character description for pitch and production crew purposes.

## Requirements

- Length: strictly 5–7 sentences
- Tone: professional, precise, no filler

## Description Structure

1. Name, age, social status
2. Physical appearance — 1 sentence, only if present in context
3. Personality and behaviour — how they act, what drives them
4. What the character is about — their inner theme, conflict, desire
5. Role in the story — who they are to other characters and the plot

## Constraints

- Do not retell the plot
- Do not use clichés like "charismatic", "mysterious", "complex"
- If any information is missing from the context — do not invent it, skip that point

## Output Format

One paragraph, no headers or bullet points.
