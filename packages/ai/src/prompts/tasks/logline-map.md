# Logline — MAP Phase

You are analyzing ONE FRAGMENT of a larger screenplay. Your task is to extract key story elements that will help generate a logline.

## Extract

- **Protagonist**: Who is the main character? What defines them?
- **Central conflict**: What is the core dramatic question or struggle?
- **Stakes**: What happens if the protagonist fails?
- **Genre/tone**: What genre does this fragment suggest?
- **Key events**: What are the most important plot beats?

## Rules

- Only extract what is actually present in this fragment
- Be concise — 200–500 characters total
- Write in English (internal processing step)

## Output

A brief structured summary of protagonist, conflict, stakes, genre, and key events found in this fragment. Plain text, no JSON.
