# Task: Short Location Description

Write a brief location description for pitch and production crew purposes.

## Requirements

- Length: 3–7 sentences
- Tone: professional, vivid, no filler

## Description Structure

1. Name and type — what it is and where it is located
2. Physical description — size, condition, key details
3. Atmosphere — mood, light, sound, the feel of the place
4. Role in the story — what happens here, what this place means to the characters

## Constraints

- Do not retell the plot
- Do not use clichés like "atmospheric", "cosy", "gloomy" without specific detail
- If any information is missing from the context — do not invent it, skip that point

## Output Format

One paragraph, no headers or bullet points.
